Movie Website

This assignment involves creating a Movie Review web application using PHP, HTML, Bootstrap, CSS, and MySQL. The project includes features for user registration, login, displaying a list of movies, and managing movie reviews. Users can register, log in, view and search movies, as well as read and leave reviews. Advanced functionalities involve user account management, sorting movies, and restricting review editing and deletion to the original users.

Portfolio App
